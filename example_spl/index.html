TodoMonkey.js Product Line

TodoMonkey.js is a product line of todo list applications implemented in javascript. The individual products are composed from a set of features using featuremonkey.js – a library to enable feature oriented software development(FOSD) with javascript.

The product line is intended as a demonstration for featuremonkey.js and not as a fully featured product line.

In reality, you might want to use a MVC library, a data persistence layer e.g. backbone.js, a template engine ... you get the picture.
For demonstration purposes, i tried to keep things simple: TodoMonkey.js's only dependencies are jquery and featuremonkey.js, which has no dependencies itself.

The feature model of the TodoMonkey.js Product Line looks like this:

Feature base is mandatory(filled circle). All other features are optional(empty circle). However, feature search depends on feature filter.

Description of individual features

base provides a basic todo list. Items can be added. Items have a label and can be marked as done.
description adds a description field to each item. Double click items so see the field.
filter adds filter functionality: show all, show done, show not done.
search adds a search field (searches all the labels).

List of products - click to run!

A product of the TodoMonkey.js SPL is generated by creating a product launcher in the form of a HTML-File. Each product launcher HTML-File includes the scripts and styles of the selected features in composition order - click on a product below and use "view source" in your browser:

  • base ... basic implementation only.
  • description · base ... with description field.
  • filter · base ... with filtering.
  • search · filter · base ... filtering and search.
  • filter · description · base ... filtering and description field.
  • search · filter · description · base ... feature creep ;)

· denotes feature composition, which is applied stepwise from right to left (right-associative operation):

d · c · b · a = d · (c · (b · a))
